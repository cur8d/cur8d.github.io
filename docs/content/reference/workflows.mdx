# GitHub Workflows

Cur8d includes pre-configured GitHub Actions workflows in `.github/workflows/`.

- **CI (`ci.yml`)**: Runs on push and PR to `main`. Includes lint, typecheck, tests, and build.
- **App Deploy (`deploy-ghpages.yml`)**: Deploys the main application to GitHub Pages.
- **Docs Deploy (`docs-ghpages.yml`)**: Deploys the Nextra documentation site to GitHub Pages.

All workflows use `mise` to ensure consistent tool versions.
